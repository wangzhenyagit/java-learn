========================================================================
动态链接库：DebugTest 项目概述
========================================================================

应用程序向导已为您创建了此 DebugTest DLL。  

本文件概要介绍组成 DebugTest 应用程序的每个文件的内容。


DebugTest.vcproj
这是使用应用程序向导生成的 VC++ 项目的主项目文件。 
它包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

DebugTest.cpp
这是主 DLL 源文件。

	此 DLL 在创建时不导出任何符号。因此，当它
	生成时不会产生 .lib 文件。如果希望此项目
	成为其他某个项目的项目依赖项，需要
	添加代码以从 DLL 导出某些符号，
	以便产生导出库，您也可以在项目“属性页”对话框中的
	“链接器”文件夹中，将“常规”属性页上的
	“忽略输入库”属性设置为“是”。

StdAfx.h, StdAfx.cpp
这些文件用于生成名为 DebugTest.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。
